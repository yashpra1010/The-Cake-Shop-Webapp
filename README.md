Cake Order Shopping Online

Developed By: -

Abstract: -
Buy a Cake is a portal which allows admin in developing counters to advertise and sell their cake. This would permit rural communities to make their wares available to the rest of the world. The objective of this project is to create a portal which would allow product information to be updated securely using a mobile device and will allow users to buy cake from the admin. The main concern is given to the village women to explore their talents and to enhance our traditional Indian culture.
